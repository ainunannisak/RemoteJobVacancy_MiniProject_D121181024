## Remote Job Vacancy 
Remote Job Vacancy is an app that provide lists of remote job in tech industries from all around the world.

## Build With
- Kotlin
- [Remotive.io API](https://remotive.io/api-documentation)
- Retrofit
- MVVM Architecture
- RecyclerView
- Glide
- WebView
